description: 로컬 서버 실행하고 게임 테스트
---

다음 단계를 실행하세요:

1. 로컬 서버 시작

   ```bash
   python -m http.server 8000
   ```

2. 브라우저에서 http://localhost:8000 열기

3. 다음 테스트 체크리스트 확인:
   - [ ] 이미지 업로드 작동
   - [ ] AI 이미지 생성 (우는 얼굴/웃는 얼굴)
   - [ ] 클릭/탭 타격 작동
   - [ ] 점수/콤보 계산 정확
   - [ ] 게임 오버 결과 이미지 표시
   - [ ] Firebase 리더보드 저장

4. 문제 발견 시 콘솔 로그 확인
